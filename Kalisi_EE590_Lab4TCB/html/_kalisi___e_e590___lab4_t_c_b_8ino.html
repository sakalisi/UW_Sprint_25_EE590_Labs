◆ scheduler()

void scheduler ( )

handles running tasks in given order

Name: scheduler

if no task is running:

  • loops through all tasks and runs the first one that is ready, given the current offset it needs to follow
  • If state is ready runs only that task and nothing else for the cycle If a task is running:
  • Calls just that task If all tasks have run, calls resetTasks() function and updates starting index

void taskA ( void * p)

Handles setup and calling of blinking LED.

Name: taskA

Parameters
parguments to any function if needed

This function is a holdover from the ICTE. Modified it. Did not change much of the underlying logic. Calls LED blinking function. if task is done, and its the current task, sets the current running task to nothing, sets current state to inactive and prints out that the task is completed.
